림컴 기본 기능인 커스텀 언어로 폰트 바꾸는 과정이 모르면 복잡하고 업데이트마다 해주는 것도 귀찮아서 프로그램으로 만듬

사용법:
다운로드 받은 폴더에서 FontManger.exe 파일을 실행한다

1.프로그램에서 게임 경로 선택을 누른 후 LimbusCompany_Data 폴더를 선택
스팀 > Limbus Company 게임 우클릭 > 설치된 파일 탭 > 로컬 파일 탐색 > LimbusCopmpany_Data 폴더 있어 경로 복사해
스팀 경로가 C 안에 제대로 있는 경우엔 자동 감지로 알아서 잡아줄 거임

2.사용 폰트 선택을 누르면 otf나 ttf 파일을 고를 수 있는 탭이 켜짐, 거기서 적용할 폰트 파일을 골라
ttf파일 추천

3.위 두 과정을 끝냈으면 폰트 변경 실행을 누르고 작업 완료 메시지가 뜰 때까지 기다려
메시지 떴으면 프로그램 꺼도 돼

4.림버스 컴퍼니를 키면 캐시 삭제 옆에 커스텀 언어 선택 버튼이 있는데 그걸 눌러서 custom 골라
만약 본래 폰트로 바꾸고 싶으면 - 고르면 돼

번외. 폰트 새로고침 버튼은 지금 폰트 유지한 채로 림컴 파일 새로고침 해주는 버튼이야. 경로만 정해주고 누르면 돼.

참고:
스킬 설명 같은 경우의 글에는 폰트가 적용이 안되는 경우가 있는데 그 때는 언어를 영어나 일본어로 사용하면 돼
업데이트할 때마다 다시 한 번 해주는 게 좋고 Windows 전용이야.
